Skiller 隐私政策

生效日期：2026-04-22 更新日期：2026-04-30 版本：1.3

欢迎使用 Skiller（以下简称"本应用"）。本应用是一款供用户浏览、搜索、收藏 Claude AI Skills 的移动应用。我们深知个人信息对您的重要性，并严格遵守 《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《中华人民共和国数据安全法》 等相关法律法规。本政策说明我们如何收集、使用、存储和保护您的个人信息。

使用本应用前请仔细阅读本政策。您开始使用本应用即视为您已阅读并同意本政策全部内容。 如您不同意本政策任一条款，请立即停止使用本应用。

一、我们收集的信息

1.1 您主动提供的信息

本应用核心浏览功能无需登录即可使用。仅当您主动选择以下功能时，我们才会收集相应账户信息：

  • 使用 GitHub 登录：我们通过 OAuth 流程获取您的 GitHub 用户名、头像 URL、公开邮箱（如有），用于在"提交 Skill"等需要身份的场景标识您。我们不会获取您的 GitHub 密码。
  • 使用 Apple 登录（Sign in with Apple）：我们获取 Apple 提供的用户唯一标识符及您选择共享的姓名/邮箱（您可使用 Apple Private Relay 转发邮箱以保护真实邮箱）。我们不会获取您的 Apple ID 密码。
  • 提交 Skill：当您从"我的 → 提交 Skill"提交 GitHub 仓库时，我们会保存您填写的仓库链接、可选邮箱、备注，以及您当前的登录用户标识，用于人工审核与可能的后续沟通。

登录与提交均为可选；您随时可在"我的"页退出登录。退出登录不会自动删除已提交的内容，如需删除请通过下方联系方式申请。

1.2 本地存储的信息

  • 收藏记录、搜索历史、主题偏好：仅存储于您的设备本地（通过 iOS SwiftData / UserDefaults），不会上传至服务器。
  • 登录会话凭证：登录后，Supabase 颁发的会话令牌通过 iOS 系统 Keychain 加密存储于您的设备，用于在应用重启后保持登录状态。卸载应用即清除。

1.3 自动收集的信息

  • 安装/浏览计数：当您查看或安装某个 Skill 时，我们会更新一个匿名的累计计数。该计数不与任何个人标识符关联。
  • 广告相关信息：为向您展示广告，我们接入的 Google AdMob SDK 可能收集设备标识（IDFV）、设备型号、操作系统版本、粗略位置（国家/地区级别）、广告交互行为等。本应用使用非个性化广告（npa=1）模式，不会用您的信息构建跨 App 用户画像。详见下方"第三方 SDK 说明"。
  • 崩溃与性能数据：我们可能收集匿名的崩溃堆栈信息以改进稳定性，不含个人身份信息。

二、我们如何使用信息

  • 向您提供 Skill 浏览、搜索、收藏等核心功能
  • 统计 Skill 的流行度（匿名聚合）
  • 展示广告以支持本应用持续运营
  • 改进产品稳定性与用户体验
  • 法律法规要求的其他用途

我们不会将您的信息用于自动化决策、用户画像或向您发送商业推广短信/邮件。

三、第三方 SDK 与服务说明

为实现特定功能，本应用集成以下第三方服务。您使用本应用即代表您同意这些第三方按其隐私政策处理相关信息：

3.1 Supabase（后端数据库与认证）

  • 服务提供方：Supabase Inc.
  • 用途：存储与分发 Skill 内容、匿名使用统计、用户身份认证（GitHub / Apple OAuth）、UGC 提交存储
  • 收集信息：匿名安装计数、请求 IP（用于防滥用，短期保留）；登录后保存 OAuth 提供方返回的用户标识、邮箱、用户名等基本信息
  • 隐私政策：https://supabase.com/privacy

3.2 Google AdMob（广告 SDK）

  • 服务提供方：Google LLC
  • 用途：在 Skill 详情页展示横幅广告
  • 收集信息：广告标识符（IDFV）、设备型号、OS 版本、粗略位置、广告点击/曝光
  • 广告模式：非个性化广告（npa=1），不基于您的兴趣画像投放
  • 隐私政策：https://policies.google.com/privacy

3.3 Sign in with Apple（Apple 登录）

  • 服务提供方：Apple Inc.
  • 用途：为您提供使用 Apple ID 一键登录的能力
  • 收集信息：Apple 提供的用户唯一标识符；姓名/邮箱由您选择是否共享，可使用 Private Relay 隐藏真实邮箱
  • 隐私政策：https://www.apple.com/legal/privacy/

3.4 GitHub OAuth（GitHub 登录）

  • 服务提供方：GitHub, Inc.
  • 用途：为您提供使用 GitHub 账号登录的能力，便于 Skill 提交流程与作者归属识别
  • 收集信息：GitHub 用户名、头像 URL、公开邮箱（如您在 GitHub 设置为公开）
  • 隐私政策：GitHub Privacy Statement

3.5 关于 App Tracking Transparency（ATT）

本应用使用非个性化广告模式，不会主动弹出 ATT 跟踪授权请求，也不会跨 App 跟踪您。您无需进行任何授权操作即可正常使用本应用。

四、信息存储与保护

  • 存储位置：后端数据存储于 Supabase 服务器（位于海外）；本地数据存储于您的设备。
  • 存储期限：匿名聚合数据长期保留；设备本地数据由您控制，您可通过卸载应用随时清除。
  • 安全措施：我们采用 HTTPS 加密传输，并对后端接口实施访问控制与防滥用限制。尽管如此，互联网环境非绝对安全，我们将尽商业合理努力保护您的信息。

五、您的权利

根据《个人信息保护法》，您对自己的个人信息享有以下权利：

  • 知情权、决定权：本政策即为我们履行告知义务的方式。
  • 查阅、复制权：本应用本地存储的数据可在应用内直接查看。
  • 删除权：卸载本应用即可清除所有本地数据。如需删除后端匿名数据，请通过下方联系方式联系我们。
  • 撤回同意：您可通过系统设置撤回广告跟踪授权；卸载应用即视为撤回对本政策的同意。
  • 投诉举报：您可通过下方联系方式向我们投诉，或向所在地网信、工信、公安等监管部门投诉举报。

六、未成年人保护

本应用主要面向成年用户。我们不会在知情的情况下收集 14 周岁以下儿童的个人信息。 若您是未成年人的监护人，发现被监护人在未经同意的情况下使用本应用，请通过下方联系方式联系我们，我们将及时采取措施删除相关信息。

七、数据的跨境传输

由于本应用后端服务（Supabase）与广告服务（AdMob）部署于中国境外，相关信息会传输至境外服务器。 我们已对境外接收方进行合理评估，并要求其提供符合法律法规要求的保护水平。您使用本应用即表示您同意此种跨境传输安排。

八、政策更新

我们可能适时更新本政策。更新后我们会通过应用内提示、更新日志或其他合理方式通知您。 若您不同意更新后的政策，请停止使用本应用；继续使用即视为接受更新后的政策。
